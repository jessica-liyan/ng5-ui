# ng5-ui for Angular5

[![npm version](https://badge.fury.io/js/ng5-ui.svg)](https://www.npmjs.com/package/ng5-ui)


#### Quick links
[Documentation, demos, and guides](https://jessica-liyan.github.io/ng5-ui/)

### Getting started

```js
npm install ng5-ui --save
or
yarn add ng5-ui
```
